# Appium Demo For iOS

## Get Start

- Install NodeJS: https://nodejs.org/en/download/package-manager

- Install Appium: https://appium.io/docs/en/latest/quickstart

```zsh
    npm install -g appium
```

```zsh
    appium setup
```

- Install Xcode

## Run Code

- Install packages:

```zsh
    npm install
```

- Run Appium:

```zsh
    appium
```

- Run create Account FB:

```zsh
    npm test
```
